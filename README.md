Tic Tac Toe
===========

Simple Tic Tac Toe web page built with HTML, CSS and JavaScript.

Files
- `index.html` — main page
- `styles.css` — styles
- `script.js` — game logic

How to run
1. Open `index.html` in a browser (double-click or right-click -> Open with).
2. Play two-player locally or check "Play vs CPU" to play against a simple computer opponent.

Controls
- "Restart Round": clears the board for a new round but keeps scores.
- "Reset Scores": clears scores and board.

Notes
- The CPU is a simple opponent: it tries to win, then block, otherwise picks a random empty cell.
- Works offline in any modern browser.
